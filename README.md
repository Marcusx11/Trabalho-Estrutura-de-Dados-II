# Trabalho-Estrutura-de-Dados-II
Trabalho de Estrutura de Dados sobre utilização de Grafos para a resolução de um problema de rotas de aviões.

O problema consistia em representar as rotas e voos entre aeroportos como um Grafo, no qual compartilha dos mesmos vértices (estes sendo os aeroportos) e 
as arestas são os caminhos disponíveis entre 1 aeroporto e o outro através de alguma rota ou voo.

O enunciado pedia para realizar diversos testes com algoritmos famosos de manipulação de Grafos como o Dijkstra para encontrar o menor caminho entre os 
aeroportos.
